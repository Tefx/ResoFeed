# Source Title and Source-Fetch Concurrency Traceability Matrix

Status: contract-only traceability artifact. This file intentionally ships no product implementation logic, schema migration, runtime route behavior, or UI rendering behavior.
Authoring step: `stc-contract-matrix-lock`.
Contract purity: enforced.

## Scope and Non-Intersection Rules

This matrix owns traceability for the accepted Source Title + bounded source-fetch concurrency delta only. It does **not** authorize product code changes, alternate schema columns, migrations, durable queues/jobs, worker/event-bus abstractions, settings/dashboard UI, or runtime behavior claims.

Global constraints preserved across every row:

- `sources.title` is the canonical display name for Source Ledger and source-listing surfaces.
- Downstream implementations must not add or require a `feed_title` storage column, response field, UI label, or compatibility alias for this delta.
- Manual per-source fetch may overlap only across different source ids; duplicate same-source fetch conflicts immediately.
- All-source manual ingest and background ingest are source-scoped bounded batches: they may overlap unrelated source fetches, skip/report busy sources, and must not enqueue persistent work.
- Source Ledger remains a flat operational ledger with text replacement states, not a progress dashboard, settings surface, queue, job history, command ledger, activity feed, or worker control panel.

## Forbidden Drift Guards

Downstream evidence must include negative proof for all of the following; these are forbidden, not optional:

- `feed_title` storage column, API/MCP response field, UI label, test fixture field, or client fallback.
- Durable queue tables, job tables, pending-work tables, ingest ledgers, activity ledgers, command-history rows, reading-history rows, durable progress records, worker processes, event buses, cross-process schedulers, sidecars, plugin registries, or settings/dashboard configuration for the parallel fetch behavior.
- Source Ledger progress bars, spinners, loaders, pulsing dots, animated ellipses, job lists, queue labels, retry panels, operation history, dashboard cards, source categories, folders, tags, drag ordering, pause/resume toggles, second URL subscription fields, or backup-management UI.
- Any transport or MCP-only product title concept unavailable to human HTTP/UI surfaces.
